# Task2

UserStory:
Как продавец, я хочу опубликовать мой товар на витрине маркетплейса, чтобы покупатели могли его увидеть и заказать.

Критерии приемки:
| Действие | Ожидаемый результат |
|-----------|-----------|
| Создание карточки товара и сохранение в черновик | Статус карточки "черновик". Карточка товара видна продавцу, не видна покупателям |
| Черновик карточки направлен на модерацию | Статус карточки "на модерации". Карточка товара видна продавцу, не видна покупателям |
| Карточка товара прошла модерацию успешно | Статус карточки "разрещена". Карточка товара видна продавцу, видна покупателям |
| Карточка товара не прошла модерацию | Статус карточки "черновик". Карточка товара видна продавцу, не видна покупателям |

User Case:
| Название поля | Действия и ожидаемый результат |
|-----------|-----------|
| Предусловие | Пользователь находится в Личном кабинете продавца маркетплейса. Магазин опубликован на маркетплейсе |
| Триггер | Публикация нового товара в магазине на маркетплейсе |
| Результат или гарантии успеха | Карточка товара в статусе "Размещена" и видна пользвоателям в открытой части маркетплейса |
| Описание (основной сценарий) | 1. Продавец формирует черновик новой карточки товара;<br>2. Продавец заполняет карточку товара в соответствии с необходимыми требованиями маркетплейса;<br>3. Сохраняет заполненную карточку кнопкой "Сохранить", статус карточки "Черновик";<br>4. Отправляет карточку на модерацию кнопкой "На модерацию", статус карточки меняется на "На модерации".<br>После успешной модерации статус карточки товара "Размещена" и видна пользователям в открытой части маркетплейса|
| Описание (альтернативный сценарий) | 1. Продавец формирует черновик новой карточки товара;<br>2. Продавец заполняет карточку товара, игнорируя обязательные требования маркетплейса (не заполнено одно или несколько обязательных полей);<br>3. Сохраняет карточку кнопкой "Сохранить", статус карточки "Черновик";<br>4. Отправляет карточку на модерацию кнопкой "На модерацию", статус картчоки меняется на "На модерации".<br>Модерация отклоняет карточку товара. Статус карточки возвращается на "Черновик", в Личный кабинет продавца поступает соответствующее уведомление со ссылкой на отклоненную карточку |
